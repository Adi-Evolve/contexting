# Contributor Covenant Code of Conduct

## Our Pledge
In the interest of fostering an open and welcoming environment, we as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism

Unacceptable behaviors include harassment, discrimination, and abusive language.

## Enforcement
Project maintainers are responsible for enforcing the Code of Conduct. If you witness or experience unacceptable behavior, please contact the maintainers at: security@adi-evolve.dev (or open a private issue).
